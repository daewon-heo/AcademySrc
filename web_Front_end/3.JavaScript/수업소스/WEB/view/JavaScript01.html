자바 스크립트 개요

스크립트 언어란?
사용자의 요구에 맞게 수행되도록 코드를 간결하게
구현하는 언어. 매우 빠르고 쉽게 배울 수 있으며
짧은 코드로 작성할 수 있도록 고안 되었다.
자바 스크립트
웹 브라우저에서 가장 많이 사용하는 인터프리터 방식의
객체지향언어. ECMA 스크립트 표준을 따르는 대표적인 웹 기술 구현 언어이다.
사용 방법
  1. Inline 방식 :
    태그 내에 직접 작성하는 방식 (간단한 소스 코드일 경우)
  2. Internal 방식 :
    script 태그를 별도로 선언하고 그 내부에 코드를 작성하여 실행하는 방식
  3. External 방식 :
    외부의 .JS 파일을 가져와 현재 웹 문서에 등록하는 방식
    (마치 CSS를 외부에서 가져 온 것 처럼 사용한다.)
주의할 점
소스코드는 함수 단위로 작성하고, 이벤트 속성을 통해 실행한다.
이벤트 속성은 태그마다 지원하는 차이가 있으며
일반적인 실행 방식은 on이벤트명 = '실행할 함수명([전달인자])'

태그에 직접 스크립트를 작성하여 구동 확인하기(Inline 방식)

scritp 태그를 활용하여 내부에서 작성한 소스코드 실행하기 (Internal 방식)
