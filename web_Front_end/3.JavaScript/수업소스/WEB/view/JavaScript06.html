문자열 + 숫자

자바와 동일하게 문자열 + 숫자 = 문자열

숫자 + 숫자 + 문자열 = (숫자 + 숫자) + 문자열 = 문자열

typeof() 함수

주어진 값의 자료형을 확인하는 함수

처음 선언 시에 자료형이 어떻게 지정되는지 확인하기가
어렵다. 그 이유는 변수명과 var 라는 동적 자료형을 기준으로
실제 값에 대한 내용을 알기 어렵기 때문이다.
따라서, 해당 변수의 실제 값에 대한 자료형을 파악하기 위한 함수가 바로 typeof()이다.
