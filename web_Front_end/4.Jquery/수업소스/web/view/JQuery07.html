$.each() 메소드

배열을 관리하는 for in 문을 메소드 형태로 제공하기 위한 함수

사용형식 :
$.each(object(배열), function(index, item){. . .}); index : 배열의 인덱스나 특정 객체의 키 값을 의미한다. item : 해당하는 인덱스나, 키의 값을 의미한다.

$('선택자').each() 메소드

사용형식 :
$('선택자').each(function(index, item){ . . . });
만약 선택한 요소가 여러 개 있을 경우 하나씩 연달아 사용할 수 있도록
반복을 제공하는 메소드이다. 매개 인자는 $.each() 메소드와 동일하다.
또한 선택자 역시 $.each()의 object와 동일하다.
