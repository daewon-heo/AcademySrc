글자 관련 스타일

color 속성 : 글자의 색상을 설정하는 속성

선택자{ color : 색상명 | 16진수숫자 | rgb값 | 
                            rgba값 | hsl값 | hsla값;}

색상값 참조 사이트 : 색상값 참조 사이트

  • 색상명으로 테스트
  • 16진수로 테스트
  • rgb로 테스트
  • rgba로 테스트
  • hsl로 테스트
  • hsla로 테스트

text-decoration 속성 : 글자의 밑줄, 취소선 등을 표현하는 속성

선택자 { text-decoration : none | underline | overline | line-through}
  • 밑줄 표현하기
  • 가운데 취소선 표현하기
  • 윗줄 표현하기
  • 하이퍼 링크 표현하기

text-transform 속성 : 영문자를 대문자 혹은
소문자, 앞글자 대문자 변경을 표현하는 속성 

선택자 { text-transform : none | uppercase | lowercase | capitalize }
  • none
  • uppercase : 영문자를 대문자로 변경
  • LOWERCASE : 영문자를 소문자로 변경
  • capitalize : 첫글자를 대문자로 변경

text-shadow 속성 : 글자에 그림자 넣기

선택자 { text-shadow : none | 가로거리 세로거리 번짐정도 색상;}
HTML5     HTML5     HTML5     HTML5

white-space : 글자 간 여백 공간 설정

선택자 { white-space : normal | nowrap | pre | pre-line;}
