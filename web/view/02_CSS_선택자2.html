속성 선택자

기본 선택자 뒤에 []를 사용하여
특정 속성과 해당 속성값을 통해 스타일을
적용할 때 사용한다.

            선택자[속성=속성값]{ . . . } 형식으로 사용한다.
            = : 속성값과 같은 때
            ~= : 특정 단어를 속성값으로 가질 때 (띄어쓰기로 구분한다.)
            |= : 특정 단어를 속성값으로 가질 때 ("" 로 구분한다.)
            ^= : 속성값이 특정 단어로 시작할 때
            $= : 속성값이 특정 단어로 끝날 때
            *= : 속성값이 특정 단어를 포함할 때

            후손 선택자 : 선조선택자 후손선택자{ . . . }

동위 선택자

동위 선택자는 동급 관계에 있는 형제 요소를 선택하고자 할 때 사용한다. 

            선택자A + 선택자B { . . . } : 바로 뒤에 있는 요소를 선택할 때
            선택자A ~ 선택자B { . . . } : 뒤의  모든 요소를 선택할 때

반응 선택자

사용자의 움직임(마우스 클릭 등)을 통해 달라지는
스타일을 적용할 때 사용한다.

            선택자:active { . . . } 사용자가 마우스 클릭 했을 때
            선택자:hover { . . . } 마우스 커서가 해당 요소 위에 올라갔을 때

상태 선택자

입력 양식에 따라 변화하는 스타일을 적용할 때 사용한다.

            input:checked{ . . . } 체크 상태 여부에 따라 스타일을 적용할 때

focus(초점)가 맞춰진 상태인지 확인하여 스타일 적용도 가능하다.
